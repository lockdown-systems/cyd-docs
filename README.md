# Cyd Docs Website

This is the docs website for https://docs.cyd.social/, which covers both [Cyd Desktop](https://github.com/lockdown-systems/cyd) and [Cyd Mobile](https://github.com/lockdown-systems/cyd-mobile).

The Cyd docs website is built using [Docusaurus](https://docusaurus.io/).

### Installation

```sh
npm install
```

### Local Development

```sh
npm run start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

### Build

```sh
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.
